Beginner

In the beginner courses, the courses mostly cover the basics of HTML(such as tables, Lists, Forms), CSS(such as flexbox, grid, basic style) and javascript (functions and events)

Intermediate

In the intermediate courses, the courses start to get more in detail. A lot of the courses have to do with learning more about CSS and also learning on using a framework such as jQuery or REACT.

Advanced

In the advanced courses, the courses go in even more detail, and here we start learning a little about backend programming, while also learning other front-end frameworks such as Vue,Angular or nodeJs.
